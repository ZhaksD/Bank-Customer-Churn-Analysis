# Bank Customer Churn Analysis

## Описание проекта
Проект направлен на анализ оттока клиентов банка (churn).  
Основная цель — выявить факторы, влияющие на уход клиентов, и предложить меры по их удержанию.

В рамках проекта выполнен полный цикл аналитики:
- загрузка и очистка данных
- исследовательский анализ
- SQL-анализ
- визуализация в Power BI
- формирование бизнес-выводов

---

## Цели анализа
- Рассчитать общий уровень оттока клиентов  
- Определить факторы, влияющие на churn  
- Выявить группы клиентов с высоким риском ухода  
- Сформулировать рекомендации для снижения оттока  

---

## Используемые инструменты
- Python (pandas, numpy, matplotlib, seaborn)  
- SQL (PostgreSQL)  
- Power BI  
- Excel  

---

## Данные
Использован датасет с информацией о клиентах банка:
- демография (возраст, страна, пол)  
- финансовые показатели (баланс, зарплата)  
- поведение клиента (активность, количество продуктов)  
- целевая переменная — факт ухода клиента (Exited)  

---

## Подготовка данных
В ходе обработки:
- удалены нерелевантные признаки: RowNumber, CustomerId, Surname  
- проверены пропуски и типы данных  
- данные приведены к корректному формату для анализа

## SQL-анализ

### Общий churn
Общий уровень оттока клиентов:
- 0.20 (20%)

Вывод:  
Каждый пятый клиент прекращает пользоваться услугами банка.

---

### Churn по возрасту

Наиболее высокий уровень оттока:
- 56 лет — 71.4%  
- 52 года — 62.7%  
- 54 года — 60.7%  
- 55 лет — 58.5%  
- 51 год — 55.5%  

Низкий уровень оттока:
- 25 лет — 3.8%  
- 19 лет — 3.7%  
- 20–30 лет — в среднем 5–10%  

Вывод:  
С увеличением возраста клиента вероятность ухода значительно возрастает.  
Наиболее проблемный сегмент — клиенты 50–60 лет.

---

### Churn по активности клиентов

- Неактивные клиенты — 26.85%  
- Активные клиенты — 14.27%  

Вывод:  
Неактивные клиенты уходят почти в 2 раза чаще.  
Активность является одним из ключевых факторов удержания.

---

### Churn по количеству продуктов

- 1 продукт — 27.7%  
- 2 продукта — 7.58%  
- 3 продукта — 82.7%  
- 4 продукта — 100%  

Вывод:  
Клиенты с двумя продуктами демонстрируют наименьший уровень оттока.  
Высокие значения при 3–4 продуктах могут быть связаны с малым количеством наблюдений.

---

## Визуализация (Power BI)

Разработан дашборд, включающий:
- ключевые показатели (churn rate, количество клиентов)  
- анализ churn по возрасту  
- анализ по активности клиентов  
- анализ по количеству продуктов  
- фильтрацию по сегментам клиентов  

---

## Основные выводы

- Общий уровень оттока составляет 20%  
- Возраст является значимым фактором: клиенты старше 50 лет уходят чаще  
- Низкая активность клиента увеличивает вероятность ухода  
- Наиболее стабильные клиенты имеют 2 продукта  

---

## Бизнес-рекомендации

- Разработать программы удержания для клиентов 50+  
- Повысить вовлечённость клиентов (уведомления, акции, персональные предложения)  
- Развивать кросс-продажи для увеличения числа продуктов  
- Улучшить пользовательский опыт и клиентский сервис
